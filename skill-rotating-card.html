CSS Animation: Rotating Card

What?

Designed using HTML and CSS this card contains information on both sides and rotates to reveal the back when the user hovers over it. The style and content on the card is fully customizable depending on the application and is setup to be easily duplicated if multiple cards are needed in the same application.

Why?

Ideal for use displaying products in a webstore, projects in a portfolio or as in the example photographs with details about the image hidden behind.

How?

If you want to learn how to create this for yourself check out my blog post where I gave a brief tutorial or get in touch to see how it could be applied to your specific needs.
